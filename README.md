# Resume Job Matching Analytics Dashboard

A data analytics project that explores candidate resumes, education, skills, experience, and hiring trends using Excel, Python, SQL, and Power BI.

## Dashboard Preview

![Resume Job Matching Dashboard](resumedasboard.png)

## Project Overview

This project analyzes candidate resume data to understand hiring patterns, education background, skills, work experience, and company preferences. The analysis was completed using Excel, Python, SQL, and Power BI to transform raw data into meaningful insights through an interactive dashboard.

## Objectives

- Analyze candidate profiles and hiring trends.
- Study education levels and university tiers.
- Compare skills, experience, and company preferences.
- Build an interactive dashboard for recruitment insights.

## Tools Used

- Microsoft Excel
- Python
- Pandas
- SQL
- Power BI

## Project Workflow

1. Collected and reviewed the dataset.
2. Cleaned and prepared the data using Python.
3. Performed SQL analysis to answer business questions.
4. Built an interactive dashboard in Power BI.
5. Generated insights from the analysis.

## Key Insights

- Most candidates have a Bachelor's degree.
- Tier 2 universities contribute the largest share of candidates.
- MNCs are the most preferred company type.
- Around 70% of candidates were marked as hired.
- Average CGPA is 7.5 with an average skill score of 14.
